# 运营封面模板工具（网页测试版）

这是从桌面压缩包改造的纯网页测试版本，可部署到 GitHub Pages。

## 使用

打开网站后可直接编辑模板；点击“AI 生图”后，在临时连接设置中填写 API 地址、API 密钥和模型。密钥只保存在当前浏览器标签页的内存中，不会写入仓库或长期保存。

API 地址按 OpenAI 兼容格式填写到 `/v1`，页面会请求 `{API 地址}/images/generations`。服务端需允许网站域名跨域访问；若返回图片 URL，该图片地址也需允许浏览器跨域读取。推荐 API 返回 `b64_json`。

## 安全说明

本版本只适合短期联调。正式部署到公司内网时，应由内网后端代理 API 请求，并将密钥保存在服务端环境变量或密钥管理系统中，不能把固定密钥放进前端页面。
